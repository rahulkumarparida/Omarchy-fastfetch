# Fastfetch Theme Collection

## Overview

This repository is a collection of **Fastfetch logos and configs**.

- The [`config/`](config) directory contains ready-to-use Fastfetch configuration themes.
- The [`logo/`](logo) directory contains logo presets and text logo files for different visual styles.
- Every config and logo directory includes a preview image so you can quickly see how it looks before using it.
